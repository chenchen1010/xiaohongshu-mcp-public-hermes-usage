# 小红书 MCP 接入说明

本仓库整理两类小红书 MCP 接入文档，分别对应普通笔记/评论采集和电商商品/店铺采集。

## 文档入口

- [小红书笔记和评论采集 MCP 接入说明](./小红书笔记和评论采集-MCP接入说明.md)：用于关键词搜索笔记、获取笔记详情、采集一级评论、采集二级回复、抓取首页推荐流和读取用户主页信息。
- [小红书电商采集 MCP 接入说明](./小红书电商采集-MCP接入说明.md)：用于创建商品、店铺或关键词电商采集任务，查询进度，读取 JSON 商品 records，并回传下游写入结果。

## 选择方式

需要笔记内容、评论洞察、二级回复或用户主页信息时，使用笔记和评论采集 MCP。

需要商品 records、店铺商品、关键词商品结果、任务进度、计费记录或 writeback 时，使用电商采集 MCP。
